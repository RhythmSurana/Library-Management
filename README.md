## 📚 Library Management System (C++)

A simple console-based Library Management System built using C++ and file handling.
This project allows an Admin to manage books and a User to borrow/return books.

## 🚀 Features : 

# 👨‍💼 Admin Panel

Add new books

Update book details

Delete books

View list of all books

# 👨‍🎓 User Panel

Borrow a book

Return a book

View list of all books

# 🛠 Technologies Used

C++

File Handling (fstream)

C-style string handling (cstring)

Console-based UI

Windows command utilities (system("cls"), dir)


## 📌 Project Structure

## ⚠ Limitations

Works on Windows (uses system("cls") and dir)

No encryption for passwords

Stores data in plain text files

Basic console UI
